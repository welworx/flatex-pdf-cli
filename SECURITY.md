# Security Policy

## Supported Versions

Only the [latest release](https://github.com/welworx/flatex-pdf-cli/releases/latest)
is supported with security fixes.

## Reporting a Vulnerability

Please report vulnerabilities privately — do **not** open a public issue:

- Preferred: [GitHub private vulnerability reporting](https://github.com/welworx/flatex-pdf-cli/security/advisories/new)
- Or email: dev@liemberger.cc

You'll get a response within a week. Since this tool processes financial
documents locally and sends nothing over the network, most issues are likely
in PDF parsing — malformed-input crashes and the like are welcome reports too.
